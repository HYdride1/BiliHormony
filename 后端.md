### 用户的URL

1. 注册：/user/signup
   客户端发送的格式为
   {
   "account" :  str
   "password": str

   }

2. 登录：/user/login
   客户端发送的格式为
   {
   "account" :  str
   "password": str

   }

3. 我的界面中的数据：/user/detail
   客户端发送的格式为
   {
   "account" :  str

   }

   服务端发送的格式为
   {
   "like": str

   "coin": int

   }
